# Contributing to SignFlow

Thank you for your interest in contributing! This document provides guidelines for contributing.

## Development Setup

1. Fork the repository
2. Clone your fork
3. Follow the [Installation Guide](docs/INSTALL.md)
4. Create a branch: `git checkout -b feature/my-feature`

## Pull Request Process

1. Ensure tests pass: `npm test`
2. Update documentation
3. Follow existing code style
4. Write clear commit messages
5. Submit PR with description

## Code Style

- JavaScript/TypeScript: ESLint + Prettier
- Vue: Single File Components
- CSS: Tailwind utility classes
- Commits: Conventional commit format

## Reporting Issues

- Use issue templates
- Provide reproduction steps
- Include environment details
- Add screenshots if applicable

## Security Issues

Please report security issues privately to security@signflow.io.

## Questions?

Join our [Discord](https://discord.gg/signflow) or email support@signflow.io.
